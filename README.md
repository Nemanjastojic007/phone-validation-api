# Phone Validation API

A Node.js API for validating phone numbers using `libphonenumber-js` and `axios`.

## Description

This is a phone validation API built with Node.js. It provides functionality to validate phone numbers using the `libphonenumber-js` library, which is a JavaScript port of Google's libphonenumber library. The API also uses `axios` for making HTTP requests.

## Installation

Install the dependencies:

```bash
npm install
```

## Dependencies

- **libphonenumber-js**: A JavaScript port of Google's libphonenumber library for parsing, formatting, and validating phone numbers
- **axios**: A promise-based HTTP client for making API requests

## Usage

This project is set up as a phone validation API. You can extend it to create endpoints for validating phone numbers, formatting them, and checking their validity.
